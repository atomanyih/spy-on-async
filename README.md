# spy-on-async

## Installation

```bash
npm install --save spy-on-async
```

## Usage

Designed to be used with jest.

### spyOnAsync

spy on a method  
```js
spyOnAsync(object, methodName)
```


### createAsyncSpy

create a spy  
```js
const spy = createAsyncSpy()
```
